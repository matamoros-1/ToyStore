# ToyStore

Se trata de un pequeño CRUD para una tienda de Juguetes, echo en la parte del Servidor(WebAPI) con .NET, C#, Entity Frenwork(migraciones y Modelos) y con incorporación de IRepository.
Por la parte del Frontend se Trabajó con angular y Bootstrap.


## Instalación

Al abrir el proyecto en la parte del servidor configurar en el archivo appsettings.json, realizar los ajustes hacia la base de datos SQL server.
En la parte del frontend en el archivo src/app/services/juguete.service.ts, En caso de ser necesario realizar el cambio de la ruta en la variable mYUrl.
